
## vuepress 创建工程

[vuepress2 快速上手](https://v2.vuepress.vuejs.org/zh/guide/getting-started.html)

```
├─ docs  // 相当于 src
│  ├─ .vuepress  // 全局的配置、组件、静态资源
│  │  └─ config  // 配置文件
│  │  │  └─ navbar.ts  // 页眉导航栏
│  │  │  └─ sidebar.ts  // 侧边栏
│  │  └─ public  // 静态资源
│  │  └─ config.ts  // 配置文件入口
│  └─ README.md  // 文档入口
├─ .gitignore
└─ package.json
```
